# const

  • 也有暂时性死去，只在当前块级作用域下有效；

  1. JavaScript 中，函数及变量的声明都将被提升到函数的最顶部。

# 本质

  1. 变量指向的是那个内存地址，对于简单数据类型，保存在变量指向的内存地址中，因此等同于常量；
  2. 对于复合类型的数据（对象，数组），变量指向的内存地址保存的是一个指针。指针是固定的，但是它指向的数据结构是可变的；

# 初始化不会提升

JavaScript 只有声明的变量会提升，初始化的不会。

    a++;
    console.log(a)
}
