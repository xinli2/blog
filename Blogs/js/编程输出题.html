# 比较规则

== 进行比较的时候，如果左右两边数据类型不一样，先转为相同数据类型，然后在进行比较

  1. {}=={} 两个对象进行比较，比较是内存地址
  2. null==undefined 相等的 / null===undefined 不想等
  3. NaN==NaN 不想等，NaN和谁都不相等
  4. [12]=='12',对象和字符串比较，把对象tostring()转换为字符串后再进行比较
  5. 剩余都是转为数字：
    • 对象==》字符串==》数字；
    • 字符串==》数字，出现一个非数字，就是NaN；
    • 布尔转数字：true==》1；false==》0
    • null==》0
    • undefined转数字是NaN
[12]==true  ==>12==1    false
[]==false   ==> 0==0    true
[]==1       ==> 0==1    false
"1"==1      ==> 1==1    true
true==2     ==> 1==2    false
